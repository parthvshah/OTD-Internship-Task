IIIT Delhi - Open Transit Data

This is a mock up for the Open Transit Data task assigned by IIIT Delhi.

Given a source and destination stop, it outputs the sequence of stops for zero hop routes or one hop routes.

The information provided is the start stop, intermediate stops and final stop with respective stop sequence which indicates the position of these stops on a particular route.

Assumption: The route information is bi-directional as no direction information was present in the given file.

Information: For a one hop route, it displays only one of the possibly many routes.

Improvement: For the different routes, the route information can be added. The text box can be fed the stops so that a user can directly enter a stop name.
